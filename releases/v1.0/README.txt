InsightClass 深见课堂 v1.0
==========================

课堂学生行为检测系统

文件说明
--------
- InsightClass-Setup.exe  安装程序（推荐）
- InsightClass.zip        便携版（解压即用）

使用方法
--------
安装版：
1. 双击 InsightClass-Setup.exe
2. 选择安装路径
3. 创建桌面快捷方式
4. 双击桌面快捷方式启动

便携版：
1. 解压 InsightClass.zip
2. 双击 InsightClass.exe 启动

功能特性
--------
- 支持 4 种行为检测：玩手机、交谈、打瞌睡、站立
- RTSP 摄像头实时监控
- 批量导入摄像头（支持海康威视 CSV 格式）
- 图片/视频检测
- Dashboard 统计

系统要求
--------
- Windows 10/11 64位
- 建议 8GB 以上内存
