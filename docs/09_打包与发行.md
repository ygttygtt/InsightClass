# 09 - 打包与发行

## 1. 打包概述

**目标**：将 Web 推理服务打包成 Windows 可执行程序，用户无需安装 Python 环境即可运行。

**输出产物**：
- **安装程序**：`InsightClass-Setup.exe`，基于 Inno Setup，包含完整安装向导
- **便携版**：`InsightClass.zip`，解压即用，适合免安装场景

## 2. 环境准备

### 2.1 基础环境

- Python 3.10+
- conda 环境（如 `QF_DL`）

### 2.2 依赖安装

```bash
# 激活 conda 环境
conda activate QF_DL

# 安装项目依赖（含 web 和 ultralytics 可选包）
pip install -e ".[web,ultralytics]"

# 安装打包工具
pip install pyinstaller cryptography
```

### 2.3 Inno Setup

下载安装 [Inno Setup 6](https://jrsoftware.org/isinfo.php)，用于生成 Windows 安装程序。

## 3. 打包流程

### 3.1 使用 PyInstaller 打包

**方式一：使用 spec 文件（推荐）**

```bash
# 打包 Web 服务版本
pyinstaller InsightClass-Web.spec

# 打包 CLI 版本（可选）
pyinstaller InsightClass.spec
```

**方式二：命令行直接打包**

```bash
pyinstaller --name InsightClass-Web ^
    --onedir ^
    --windowed ^
    --exclude-module PyQt5 ^
    --add-data "configs;configs" ^
    --hidden-import insightclass ^
    --hidden-import insightclass.web ^
    launcher.pyw
```

### 3.2 复制补充文件

打包完成后，需手动复制以下文件到 `dist/InsightClass-Web/` 目录：

```bash
# 摄像头配置（用户自定义）
copy configs\cameras.yaml dist\InsightClass-Web\configs\

# 应用配置
copy configs\app.yaml dist\InsightClass-Web\configs\
```

### 3.3 生成 ZIP 便携版

```powershell
# 在 dist 目录下压缩
Compress-Archive -Path dist\InsightClass-Web -DestinationPath releases\v1.0\InsightClass.zip
```

### 3.4 生成安装程序

1. 用 Inno Setup 打开 `installer.iss`
2. 点击 **Build > Compile**
3. 输出文件：`releases\v1.0\InsightClass-Setup.exe`

## 4. 关键配置文件

### 4.1 launcher.pyw

启动器入口，功能：
- 检测端口占用并提示
- 启动 FastAPI 服务
- 带 tkinter 控制窗口（启动/停止/打开浏览器）
- 支持后台运行，关闭窗口可选择最小化到托盘或退出

### 4.2 InsightClass-Web.spec

PyInstaller 配置，关键参数：
- `--exclude-module PyQt5`：避免与 PySide6 冲突
- `--hidden-import`：确保动态导入的模块被包含
- `--add-data`：打包配置文件目录

### 4.3 installer.iss

Inno Setup 脚本，配置：
- 安装目录、快捷方式
- 卸载程序
- 开机自启动（可选）

## 5. 常见问题与解决

### 5.1 PyQt5 和 PySide6 冲突

**现象**：打包后启动报错 `qt.qpa.plugin: Could not load the Qt platform plugin "windows"`

**解决**：在 spec 文件或命令行中排除 PyQt5：
```bash
--exclude-module PyQt5
```

### 5.2 PySide6 WebEngine 打包慢/体积大

**现象**：打包时间超过 10 分钟，产物体积超过 500MB

**解决**：项目已改用系统浏览器方案（`webbrowser.open()`），不再打包 WebEngine。如仍遇到，确认 spec 文件中没有包含 PySide6-WebEngine 相关模块。

### 5.3 文件被占用错误

**现象**：打包或覆盖文件时提示 `PermissionError: [WinError 32] 另一个程序正在使用此文件`

**解决**：
```powershell
# 终止所有相关进程
taskkill /F /IM InsightClass-Web.exe
taskkill /F /IM python.exe
```

### 5.4 端口被占用

**现象**：启动服务时提示 `Address already in use`

**解决**：
```powershell
# 查找占用端口的进程（默认端口 8080）
netstat -ano | findstr :8080

# 终止进程（替换 <PID>）
taskkill /F /PID <PID>
```

### 5.5 杀毒软件误报

**现象**：打包的 exe 被 Windows Defender 或其他杀毒软件拦截

**解决**：
- 在杀毒软件中添加排除目录
- 使用代码签名证书签名（正式发行时）

## 6. 发行版目录结构

```
releases/
└── v1.0/
    ├── InsightClass-Setup.exe    # 安装程序
    ├── InsightClass.zip          # 便携版
    └── README.txt                # 版本说明
```

### README.txt 示例内容

```
InsightClass v1.0
================

课堂学生行为检测系统

安装方式：
1. 运行 InsightClass-Setup.exe 按向导安装
2. 或解压 InsightClass.zip 到任意目录

启动方式：
- 安装版：桌面快捷方式或开始菜单
- 便携版：运行 InsightClass-Web.exe

默认访问地址：http://localhost:8080
```

## 7. 自动化构建脚本

使用 `scripts/build_package.ps1` 一键构建：

```powershell
# 完整构建（打包 + 便携版 + 安装程序）
.\scripts\build_package.ps1

# 仅打包
.\scripts\build_package.ps1 -PackageOnly

# 指定版本号
.\scripts\build_package.ps1 -Version "1.1.0"
```

脚本流程：
1. 清理旧的 `build/` 和 `dist/` 目录
2. 运行 PyInstaller
3. 复制补充配置文件
4. 生成 ZIP 便携版
5. 调用 Inno Setup 生成安装程序
6. 输出到 `releases/<version>/` 目录
